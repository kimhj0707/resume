## 3. 대표 프로젝트

### RealWalk | GIS 기반 정밀 상권 분석 서비스
- **한 줄 소개:** 보행·입지 데이터를 활용한 상권 분석 지도 서비스
- **문제 정의:** 기존 상권 분석 서비스는 행정구역·직선거리 중심으로 실제 상권 특성을 반영하기 어려움
- **내 역할:** 서비스 기획, 데이터 요구사항 정의, 분석 구조 설계, MVP 화면 흐름 기획
- **해결 방식:**  
  - 건물·POI·보행 데이터를 결합한 분석 시나리오 설계  
  - 지도 기반 시각화 구조 기획
- **결과:**  
  - 금천구 기준 건물 약 15,000개 데이터 적재  
  - POI 수집 자동화 구조 설계
- **링크:**  
  - GitHub: https://github.com/kimhj0707/realwalk

- **해결 방식:**  
  - 추천 로직 입력·출력 정의  
  - 관리자 운영을 고려한 구조 분리
- **결과:**  
  - 마이크로서비스 구조 기반 설계 경험  
  - 발표용 기획·구조 문서 작성
- **링크:**
