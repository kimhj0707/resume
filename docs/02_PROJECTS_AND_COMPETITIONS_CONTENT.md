### Project 1. K-푸드 수출 패스포트 AI

한 줄 요약:
식품 라벨 이미지를 기반으로  
수출 규정 검증과 홍보 문구 생성을 자동화한 AI 서비스

문제:

- 국가별 수출 규정 확인이 복잡
- 소규모 제조사의 실무 부담 큼

역할:

- 전체 서비스 기획
- OCR → 규정 체크 → LLM 파이프라인 설계
- MVP 범위 정의

해결:

- OCR 기반 텍스트 추출
- 규정 JSON 룰 구조화
- 자동 위반 체크
- LLM 홍보 문구 생성

결과:

- 이미지 업로드 기반 데모 구현
- 공모전/발표용 MVP 완성

---

### Project 2. 일드림 (Il-Dream)

한 줄 요약:
공고 이전 신호를 기반으로  
채용 가능성을 예측하는 AI 일자리 매칭 서비스

문제:

- 채용 정보는 항상 늦게 공개됨
- 구직자는 타이밍에서 밀림

역할:

- 문제 정의 및 구조 설계
- 예측 개념 모델 설계
- 서비스 스토리라인 구성

해결:

- 투자/조직 변화/과거 채용 이력 등 신호 활용
- 채용 가능성 분류 개념 제안

결과:

- Tech for All 공모전 최우수상 수상
- 대표 기획 프로젝트로 활용
