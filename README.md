## 👋 소개

저는 **개발 구조를 이해하고, 기술 기반으로 서비스를 설계하는 IT 서비스 기획자**입니다.  

단순 아이디어 제안이 아니라,
- 실제 구현 가능한 구조인지
- 데이터는 어디서 오고 어떻게 흐르는지
- 개발자가 무엇을 어려워하는지

를 기준으로 **기획 → 구조 설계 → 화면 흐름 → 기술 선택까지 연결하는 기획**을 지향합니다.

현재 다수의 **공공·기술 기반 공모전**에 참여하며  
기획서, 제안서, 발표자료를 직접 작성하고 프로젝트 구조를 설계했습니다.
